# 日本語執筆スタイルガイド

## 第一文（定義文）
- 記事の第一文で対象を定義する。
- 基本形は「Xは、Yである。」を用いる。
- 第一文で対象の種類・時代・文脈が判別できる状態にする。

## 導入部
- 導入部は評価より定義を優先する。
- 何者か／いつの話か／なぜ重要かを短く提示する。
- 導入部だけで記事要旨が読める情報密度を目標にする。

## 節構成
- 見出し階層で論点を分離し、長文でも目次で走査可能にする。
- 箇条書きは補助として使用し、説明の主軸は段落で記述する。
- 話題転換は段落分割または見出しで行う。

## 日本語表記ルール
- 文体は常体（だ・である調）を基本とする。
- 宣伝調・煽り調・感想調を避け、中立で説明的な語り口を維持する。
- 断定しにくい記述は「〜とされる」「〜とみられる」などで根拠との距離を示す。
- 重要語の内部リンクは初出中心に付与し、同一段落で過密にしない。
- 用語表記はページ内で統一し、表記ゆれを防ぐ。
