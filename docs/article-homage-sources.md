# 個別記事オマージュ元対応表

このメモは、公開ページに直接掲載しない制作・編集用の対応表である。実在Wikipediaの記事を「文章の複写元」ではなく、見出し構成、情報の粒度、infobox項目、批判・脚注の置き方を観察するための参照先として扱う。

## 参照方針

- 実在Wikipediaから本文を転載しない。参照するのは記事構造、見出し順、情報の切り分け方、脚注・関連項目の配置に限る。
- 1記事につき「主オマージュ元」1本と「補助参照」1〜3本を決める。
- Tenzu側の記事では、作中の公開資料として自然な表現に置き換え、実在名・実在事件との直接対応を断定しない。
- 404などのユーティリティページは記事化せず、オマージュ対象から外す。
- 候補は日本国内・海外、過去・現代の両方から複数持ち、1つの記事へ単純対応させない。最終的な記事は、複数の参照型を合成してTenzu側の公開資料らしい構成にする。

## 次回の確認事項

- 企業記事は「沿革」を厚くするか、「現在の事業・社会的影響」を厚くするか。過去企業（財閥・東インド会社・クルップ）をどの程度混ぜるか。
- 人物記事は「来歴」を厚くするか、「政策・活動・評価」を厚くするか。日本現代政治家型だけでなく、世界の危機管理型指導者も使うか。
- 政府記事は「国家記事」ではなく「政権・行政機構記事」として書く方針でよいか。分断国家・戦間期国家・冷戦国家の型をどれだけ入れるか。
- 佐渡島組は暴力団記事に強く寄せるか、港湾物流団体として控えめに書くか。世界の組織犯罪記事を参照する場合も、公開資料らしい慎重表現を維持する。
- 各記事の主オマージュ元は1本に固定するか、「主1本 + 日本過去1本 + 世界過去1本 + 世界現代1本」の4本セットで運用するか。
