# 天通辞典 編集方針案

この文書は、天通辞典の記事本文とメインページを整えるための編集基準である。実在のオンライン百科事典が採る基本方針を参照し、天通辞典の内部制度として自然に見える形へ置き換える。

## 基本理念

天通辞典は、公開資料に基づき、人物、組織、行政、企業活動、社会制度を百科事典形式で整理する公開資料網である。

記事は宣伝、感想、制作上の説明ではなく、確認可能な情報を中立的に記述することを目的とする。

## 五つの基本原則

### 1. 天通辞典は百科事典である

天通辞典は速報欄、社説欄、宣伝媒体、単なる人物一覧ではない。記事は対象を定義し、来歴、制度、組織、事業、影響、評価、関連資料を整理する。

### 2. 中立的な観点

記事は特定の政府、企業、団体、人物の立場を代弁しない。複数の立場がある事項では、行政発表、企業発表、報道、研究、証言、外部推計を区別し、評価語を本文の断定として置かない。

### 3. 検証可能性

記事に掲載する情報は、公開資料、行政文書、企業開示、報道、研究、演説記録、統計、関係者証言などによって確認できる必要がある。

確認できない噂、断片的な証言、編集者の推測は本文の根拠にしない。

### 4. 独自研究の禁止

公開資料にない因果関係、動機、密約、将来の結果を編集者が推論して断定してはならない。複数資料を組み合わせて新しい結論を作る場合も、その結論自体を述べた資料がない限り断定しない。

### 5. 礼節と合意形成

編集は記事の改善を目的とする。対立する記述は、出典、表現、掲載位置、重みづけを確認しながら調整する。

## 天通辞典ではないもの

天通辞典は、次の目的では使わない。

- 宣伝や広報文の転載
- 速報情報の無制限な集積
- 未確認情報の掲載
- 個人攻撃や風評の拡散
- 将来予測や願望の発表
- 制作上の説明、舞台設定解説、作品外説明

## 資料区分

記事内の根拠は、可能な限り次の区分を意識する。

| 区分 | 例 | 扱い |
|---|---|---|
| 行政資料 | 法令、白書、統計、公報 | 制度や数値の基礎資料として扱う |
| 企業資料 | 沿革、報告書、発表資料 | 企業側の説明として扱い、宣伝調にしない |
| 報道資料 | 新聞、通信社、放送、調査報道 | 出来事や社会的評価の資料として扱う |
| 研究資料 | 論文、研究報告、専門書 | 背景説明と評価の重みづけに用いる |
| 証言資料 | インタビュー、回想、関係者証言 | 主観的資料として距離を取って用いる |
| 推計資料 | 民間調査、外部推定 | 数値を断定せず、推計であることを明記する |

## 人物記事の基準

存命人物、現職者、企業経営者については特に慎重に扱う。批判、疑惑、健康状態、家族、私生活、思想、派閥、人間関係は、信頼できる公開資料がある場合に限って掲載する。

人物記事では、関係性を単独節として過剰に整理せず、来歴、職務、政策、活動、評価の中で必要な範囲に留める。

## 未来事項の扱い

公開時点で発生していない出来事、後年の評価、後日の結果は掲載しない。

予定、予測、計画については、公表資料がある場合に限り「予定されている」「検討されている」「報じられている」として扱う。

## 表現上の禁則

公開記事では、次のような制作上の語を用いない。

- 本作
- 作中
- 設定
- 世界観
- 動画
- シナリオ
- 物語上
- 読者向け
- 作者
- 元ネタ

必要な場合は、天通辞典内の自然な語に置き換える。

| 避ける表現 | 置き換え例 |
|---|---|
| 作中社会 | 社会一般、公開資料上、当時の報道空間 |
| 設定 | 制度、経緯、公開資料上の説明 |
| 本編開始前 | 公表時点、当該資料の基準時点 |
| 物語上の役割 | 公的役職、社会的影響、報道上の扱い |
| キャラクター | 人物 |

## 洩八グループ表記

西日本側の生活基盤財閥については、今後の標準表記を「洩八グループ」とする。

既存ページ・内部メモ・検索インデックスに残る「守矢グループ」は、必要に応じて旧称、関連語、または削除対象として整理する。八坂神奈子個人記事では、肩書きを「洩八グループ会長」とする。

## 実装時の優先順位

1. 生成元スクリプトの表記を修正する。
2. 生成済みHTMLを再生成する。
3. 検索インデックスを更新する。
4. 内部メモの表記を更新する。
5. 公開記事本文の方針ページ化を検討する。
