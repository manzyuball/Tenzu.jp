# 財閥三社 画像・図式参照計画

このメモは、元となるWikipediaページに実際に掲載されている画像・図式の種類を確認し、Tenzu記事で同種の画像・図式だけを用意するための内部メモである。公開ページへは掲載しない。

## 基本方針

- 元Wikipediaページに存在する画像・図式の「種類」だけを採用する。
- 実在企業のロゴ、社屋写真、組織図をTenzu記事へそのまま流用しない。必要な場合は、ライセンス確認済みのWikimedia Commons画像を参考資料として扱い、Tenzu用には既存素材または新規作成した別画像を使う。
- ロゴ、社屋、組織図、通信網図、事業系統図のうち、元ページに見られるものだけを候補にする。
- 元ページにない種類の画像、例えばキャラクター絵、宣伝バナー、未来的なコンセプトアートは企業記事には入れない。

## 天通の画像・図式候補

参照ページで確認した画像・図式の例:

| 参照ページ | 確認した画像・図式 | Tenzu側で対応させる候補 |
| --- | --- | --- |
| KDDI | 企業ロゴ、auロゴ、本社・オフィスビル写真、関連施設写真 | 既存 `tentsu_logo.png`、既存 `tentsu_hq_building*.png`、通信拠点写真風画像 |
| ソフトバンクグループ | 企業・サービスロゴ、本社ビル写真、通信史図 | 天通グループロゴ、天通本社、天網中継塔または通信史の模式図 |

| Meta Platforms | 企業ロゴ、本社・キャンパス系画像、サービス系図版 | SNS・EC系子会社の位置づけ図、データセンター/拠点画像 |

## 天通で次に作る画像案

1. infobox画像: 既存 `assets/images/tentsu_logo.png` を継続使用。
2. 本社画像: `assets/images/tentsu_hq_building.png` または `assets/images/tentsu_hq_building_v2.png` を、KDDI/ソフトバンク系記事の本社写真と同じ位置づけで使用。
3. 組織図: Alphabet記事の組織図型に合わせ、天通グループの中核会社から通信系子会社、報道系子会社、プラットフォーム系子会社、資料網・社会貢献部門、計算基盤系子会社へ分かれる図をSVGで作る。
4. 通信網図: ソフトバンク系の通信史・ネットワーク図に合わせ、天網中継塔ネットワークを模式図として作る。ただし元ページに近い図式に限定し、SF的なイラストにはしない。

## 追加でユーザーへ確認すること

- KDDI/ソフトバンク記事のように、infoboxはロゴのみでよいか。本社写真も入れるか。
- Alphabet型の組織図を天通記事に追加してよいか。
- 天通グループの系列子会社名を正式に決めるか、当面は「通信系子会社」などの総称にするか。
- 元Wikipediaページの画像に似せる場合、写実写真風ではなく、Commons掲載写真のような説明的・記録的な画調でよいか。
