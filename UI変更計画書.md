# UI変更計画書

## 0. 目的

Tenzu.jp / 天通辞典のUIを、従来の「架空世界の設定資料集」風から、実在する日本型ネット百科サービス風へ変更する。

基礎はWikipedia型の記事構造とする。ただしトップページは、日本のポータルサイトやネット辞典にありがちな、検索欄、ニュースフィード、急上昇ワード、掲示板、広告、カテゴリ導線が混在した雑然とした画面にする。

目指す印象は次の通り。

- Wikipediaの百科構造
- ニコニコ大百科の掲示板・単語記事感
- Yahoo! JAPAN風のポータル導線
- 日本の無料サイトにありがちな広告の多さ
- 実在サイトのような生活ノイズ

重要なのは、トップページで世界の核心語や政治・軍事設定を説明しないこと。トップページは、普通の利用者が毎日だらだら見るネット辞典の入口にする。真面目で精密な記事は検索やカテゴリの奥に置く。

---

## 1. 現状の問題点

### 1.1 設定資料集に見える要因

以下の要素をトップページに出しすぎると、実在する百科サイトではなく、架空世界の説明ページに見える。

- 国家名、政府名、軍事組織名をトップに並べる
- 企業・人物・地理・軍事を「注目記事」として前面に出す
- 天通内部の部署名や資料提供元を目立たせる
- 世界の核心語を急上昇ワードに入れる
- トップページが整然としすぎている
- 広告が真面目すぎる
- フィードが政治・軍事・企業寄りすぎる

実在する日本のネットサービスでは、トップに出る情報はもっと雑で、生活臭く、低俗で、広告に汚れている。

### 1.2 削除対象

以下のリアルタイム性を要求する要素は、静的サイトでは破綻しやすいため削除する。

- 今日の日付表示
- 現在時刻
- 今日の天気
- 今日は何の日
- 今日の運勢
- リアルタイム閲覧数
- 「現在」「本日」「速報中」など、自動更新が必要に見える表記

静的に置く場合も、日付固定は避ける。古くなると廃墟サイトに見えるため。

## 2. 全体デザイン方針

### 2.1 配色

洗練されすぎたSaaS風ではなく、少し古い日本のポータルサイト風にする。

```css
:root {
  --bg: #f5f5f5;
  --panel: #ffffff;
  --line: #d5d5d5;
  --line-dark: #b8b8b8;
  --text: #222222;
  --muted: #666666;
  --link: #0645ad;
  --blue: #1b3f8b;
  --blue-bg: #f3f6fb;
  --red: #d60000;
  --orange: #f0a000;
  --ad-yellow: #fff8d8;
}
```

### 2.2 基本文法

- 背景は白または薄灰
- リンクは濃い青
- 枠線は細い灰色
- 見出しは薄青・薄灰の帯
- 大きな角丸や影は控える
- 余白はやや狭め
- 情報密度は高め
- 日本語リンクが多い画面にする

### 2.3 画面密度

トップページは「整理された企業サイト」ではなく、「機能を足し続けたネット辞典」にする。

画面上に次の要素が同時に見えてよい。

- 検索欄
- 左メニュー
- 最新ニュース
- 注目記事
- 最近更新された記事
- 新規作成記事
- 急上昇ワード
- 掲示板で話題の記事
- 広告
- 人気カテゴリ

## 3. トップページUI

### 3.1 全体構成

```text
┌──────────────────────────────────────────────┐
│ 天通ポータル｜ニュース｜辞典｜動画｜生放送｜ゲーム｜占い｜ショッピング │
├──────────────────────────────────────────────┤
│ Tenzu.jp 天通辞典       [検索カテゴリ▼][検索欄][検索] │
│ 単語記事｜人物｜組織・団体｜出来事｜地域｜作品｜ニュース｜掲示板 │
├────────┬───────────────────────┬────────────┤
│ 左メニュー │ 中央メイン                  │ 右カラム       │
│            │ ようこそ                    │ 急上昇ワード   │
│            │ 最新ニュース                │ 広告           │
│            │ 注目の記事                  │ 掲示板で話題   │
│            │ 最近更新された記事          │ 広告           │
│            │ 新規作成記事                │ 新着記事       │
│            │ 人気カテゴリ                │ 広告           │
└────────┴───────────────────────┴────────────┘
```

### 3.2 ヘッダー

上部には日本型ポータル導線を置く。

```html
<div class="portal-bar">
  <a href="#">天通ポータル</a>
  <a href="#">ニュース</a>
  <a href="#">辞典</a>
  <a href="#">動画</a>
  <a href="#">生放送</a>
  <a href="#">ゲーム</a>
  <a href="#">占い</a>
  <a href="#">ショッピング</a>
  <a href="#">もっと見る</a>
</div>
```

「動画」「生放送」「ゲーム」「占い」「ショッピング」は、ネット辞典には本来不要な要素だが、日本のポータルサイトらしい無駄として残す。

### 3.3 ロゴ・検索欄

```html
<header class="site-header">
  <div class="brand">
    <span class="brand-main">Tenzu.jp</span>
    <span class="brand-sub">天通辞典</span>
    <span class="brand-copy">みんなでつくるネット百科事典</span>
  </div>

  <form class="search-box">
    <select>
      <option>すべて</option>
      <option>単語記事</option>
      <option>人物</option>
      <option>組織・団体</option>
      <option>出来事</option>
      <option>地域</option>
      <option>作品</option>
      <option>掲示板</option>
    </select>
    <input type="search" placeholder="天通辞典を検索">
    <button type="submit">検索</button>
  </form>
</header>
```

検索欄は大きくする。トップページの主機能は検索であり、ここはYahoo! JAPAN寄りにする。

### 3.4 マスコット

中央の「ようこそ」欄には、白い鳥型マスコットを置く。

仕様:

- 白い丸い鳥
- 虫眼鏡を持つ
- 本を読んでいる
- 赤い小さな帽子をかぶる
- 帽子は射命丸文の帽子を連想させる形
- ただし露骨なキャラクター化は避ける
- 羽根や天狗要素は必要最小限

これは企業・サービスの公式マスコットとして扱う。人物欄ではない。

### 3.5 ようこそ欄

```text
天通辞典へようこそ

天通辞典は、単語・人物・組織・出来事・地域・作品などについて、意味や関連情報をまとめる辞典・百科サービスです。
記事は利用者の編集、公開資料、報道記録などをもとに随時更新されます。

天通辞典を知る　編集ガイド　よくある質問
```

ここで国家・軍事・境界線・企業設定を出さない。

### 3.6 最新ニュース欄

ニュース欄は生活ノイズにする。設定資料集的な固有語を出しすぎない。

例:

```text
最新ニュース

[総合] [国内] [経済] [エンタメ] [スポーツ] [IT] [地域] [ライフ]

・中日、延長戦でサヨナラ負け　監督「切り替えるしかない」
・人気俳優の近影に反響　「雰囲気変わった？」
・岐阜県の道の駅で行列　限定ソフトが話題に
・元アイドル、結婚報告に祝福の声
・「冷凍うどん」人気ランキング　意外な1位は
・都内で突然の雷雨　帰宅時間帯に影響
・議員の過去投稿が再拡散　SNSで賛否
・YouTuberの謝罪動画にコメント殺到
```

「境界線近く」ではなく「岐阜県」のような普通の地名で書く。

### 3.7 急上昇ワード

急上昇ワードは、重要設定語ではなくネットミーム・俗語・くだらない記事名中心にする。

```text
急上昇ワード

1 エショワが消された写真
2 プロリダマン
3 タクシードライバー
4 鹿さんやりますねぇ！
5 強い海老リンク
6 岐阜の例の看板
7 泣き崩れる嫁
8 フロリダマン
9 米だけ炊いて寝た人
10 例の謝罪文
```

政治・軍事・企業・政府の固有名詞は原則入れない。

### 3.8 広告欄

広告は実際の日本のWeb広告のように、安っぽく、写真素材風、詐欺臭く、生活臭くする。

方向性:

- 薄毛・抜け毛
- ダイエット
- 美容・整形
- 自然派健康食品
- 謎のスマホゲーム
- ピン抜きゲーム
- 基地発展ゲーム
- 性格診断
- 投資・副業風の怪しい診断

広告画像はAI絵っぽくしない。実装では、以下のような方針が望ましい。

- 実写広告風のレイアウト
- 人物写真風の汎用素材
- 商品写真風の小物
- 大きなCTAボタン
- `[PR]` `[広告]` 表記
- 緑、赤、黄色のボタン
- 過剰なキャッチコピー

例:

```text
[PR] 最近、髪のボリュームが気になりませんか？
専門家が教える、見た目年齢の新常識
詳しくはこちら
```

```text
[広告] ピンを抜いて王様を助けよう！
今すぐ無料でプレイ
```

```text
[PR] 飲むだけでスッキリ？
話題の自然派サポート習慣
初回限定 980円
```

```text
[広告] 美女司令官と基地を発展！
今なら10連ガチャ無料
今すぐプレイ
```

広告は本筋と関係なくてよい。むしろ無関係である方が実在感が出る。

### 3.9 掲示板で話題の記事

```text
掲示板で話題の記事

・鹿さんやりますねぇ！
  214件のコメント

・例の謝罪文
  188件のコメント

・岐阜の例の看板
  132件のコメント

・プロリダマン
  98件のコメント

・強い海老リンク
  76件のコメント
```

コメント欄はくだらなくする。議論ではなく、ネットの雑談・突っ込み・編集合戦感を出す。

## 4. 個別記事ページUI

### 4.1 基本方針

個別記事ページはトップページよりWikipedia寄りにする。

構成は次の通り。

- Wikipedia型の記事本文
- ニコニコ大百科型の掲示板
- 右側の質の悪い広告
- 左側のメニュー
- 上部の検索欄

トップほどポータル色を強めない。記事ページでは本文の読みやすさを優先する。

### 4.2 全体構成

```text
┌──────────────────────────────────────────────┐
│ 天通ポータル｜ニュース｜辞典｜動画｜ゲーム｜ヘルプ │
├──────────────────────────────────────────────┤
│ Tenzu.jp 天通辞典        [検索欄] [検索]        │
├──────────────────────────────────────────────┤
│ 単語記事｜人物｜組織・団体｜出来事｜地域｜作品｜掲示板 │
├────────┬───────────────────────┬────────────┤
│ 左メニュー │ 記事本文                  │ 右広告       │
│            │ タイトル                  │ 広告         │
│            │ 記事/ノート/編集/履歴      │ 急上昇ワード │
│            │ 導入文                    │ 新着記事     │
│            │ 目次                      │ 広告         │
│            │ 本文                      │              │
│            │ 脚注                      │              │
│            │ 関連項目                  │              │
│            │ カテゴリ                  │              │
│            │ 掲示板                    │              │
└────────┴───────────────────────┴────────────┘
```

### 4.3 記事上部

```text
タクシードライバー

出典: 天通辞典

[記事] [ノート]                         [閲覧] [編集] [履歴表示]
```

記事名は明朝体寄りで大きくする。本文は通常のゴシックでもよい。

### 4.4 注意テンプレート

```text
この記事には複数の問題があります。
- 出典が不足している記述があります。
- 独自研究を含む可能性があります。
- 掲示板での議論を本文へ反映する際は、編集方針を確認してください。
```

注意テンプレートはWikipedia風に、薄黄色または薄灰の枠で表示する。

### 4.5 Infobox

記事本文右上に、Wikipedia風のinfoboxを置く。

```text
┌──────────────┐
│ タクシードライバー │
├──────────────┤
│ 原題　Taxi Driver │
│ 公開　1976年      │
│ 監督　M.スコセッシ │
│ 主演　R.デ・ニーロ │
│ 分類　映画        │

```

右広告カラムとは別に、本文内でfloat rightする。

### 4.6 目次

```text
目次
1 概要
2 あらすじ
3 評価
4 日本での受容
5 インターネット上での扱い
6 脚注
```

例:

```text
タクシードライバーは、1976年に公開されたアメリカ合衆国の映画である。
公開後、映画史上の重要作品として扱われる一方、インターネット上では断片的な引用や画像素材としても広く流通している。

概要
本文。

あらすじ
本文。

評価
本文。

日本での受容
本文。

インターネット上での扱い
本文。

脚注
出典一覧。

関連項目
関連リンク。
```

実在記事、架空記事、ネットミーム記事を同じUIに入れる。

### 4.8 掲示板

記事下部に掲示板を置く。

```text
掲示板
この掲示板は過去ログ表示です。新規投稿機能は現在停止しています。

1 ななしの利用者
この記事、概要より掲示板の方が詳しいの草

2 ななしの利用者
ソース貼れ

3 ななしの利用者

古い版の方が読みやすかった

5 ななしの利用者
この記事名で検索すると広告しか出てこないの終わってる
```

静的サイトなので、投稿機能は実装しない。過去ログ表示として見せる。

## 7. 禁止・抑制事項

### 7.1 トップページに出さないもの

トップページでは以下を原則として出さない。

- 世界の核心語
- 軍事・政府・企業の重要項目
- 天通内部部署名
- 配信元表記
- 資料提供元表記
- 具体的な設定説明
- 本編・作中・設定・世界観などのメタ語

### 7.2 リアルタイム風UIの禁止

以下は削除する。

- 今日は何の日
- 今日の運勢
- 今日の天気
- 現在時刻
- 現在の日付
- リアルタイム閲覧数

### 7.3 広告の注意

実在企業名や実在サービス名を使わない。広告は架空名にする。

ただし見た目は実在広告風にする。

## 10. 最終イメージ

トップページは、検索欄と百科導線を中心にしつつ、ニュース、急上昇、掲示板、広告が過剰に混じった日本型ネットポータルにする。

個別記事ページは、Wikipedia風の本文構造を中心にしつつ、右側に広告、下部に掲示板を置く。
